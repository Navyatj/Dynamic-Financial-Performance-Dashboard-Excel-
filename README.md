# Dynamic-Financial-Performance-Dashboard-Excel-
This project involves building a dynamic financial performance dashboard using Microsoft Excel to analyze revenue, expenses, and profitability across multiple years and departments.The dashboard enables interactive analysis through slicers and PivotTables, allowing users to filter data by year and department and instantly view updated financial insights.

The project focuses on transforming structured financial data into meaningful visual insights for decision-making, highlighting trends, department-wise performance, and overall financial health.
